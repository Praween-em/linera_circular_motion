Physics of Motion

Motion can be described as the change in position of an object over time. Here we compare two fundamental types of motion:

Linear Motion

Motion along a straight line, described by:

s = s₀ + v₀t + ½at²

Where s is position, v₀ is initial velocity, a is acceleration, and t is time.

In linear motion, velocity changes uniformly with acceleration, and position changes quadratically with time when acceleration is constant.

Circular Motion

Motion along a circular path, described by:

θ = θ₀ + ωt

Where θ is angular position, ω is angular velocity, and t is time.

In uniform circular motion, the speed is constant but velocity changes direction continuously, resulting in centripetal acceleration toward the center:

a = v²/r = ω²r
